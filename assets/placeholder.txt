## Usage Instructions

1. Add the actual image files to this directory
2. Ensure proper naming convention
3. Recommended image formats: PNG for icons, JPG for photos
4. Optimize images for mobile (compress without losing quality)

## Image Guidelines

- Use high-quality images
- Maintain consistent aspect ratios
- Optimize for fast loading
- Use appropriate file sizes (under 500KB per image)
- Test images on both iOS and Android
